O jogo consiste em escapar da sala através de uma chave.
O jogador controla o personagem azul através das setas do teclado. Ele deve pegar a chave (quadrado verde) para que a porta (retângulo amarelo) seja aberta.
O jogador só pode sair da sala após pegar a chave.
Caso o jogador colida com algum inimigo (personagens vermelhos), o jogo se encerra.

Na tela inicial, o jogador pode escolher o modo de jogo:

Easy - 3 inimigos
Medium - 5 inimigos
Hard - 7 inimigos
